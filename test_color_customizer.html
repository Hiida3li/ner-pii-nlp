Features

  • Customizable colors for each entity type
  • 5 preset color schemes (Default, High Contrast, Colorblind Safe, Dark Mode, Pastel)
  • Individual color reset buttons
  • Export/Import color configurations
  • Colors saved in localStorage for persistence
  • Real-time color updates
  • Accessibility-focused design
  • Collapsible panel to save space
