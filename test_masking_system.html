🔒 Bidirectional PII Masking System Test

📋 Test Workflow

Step 1: Original Text Analysis

Enter text with PII entities → System analyzes and creates mappings

Step 2: Get Masked Text

Original text → Masked text (for sending to LLM)

Example: "احمد يعمل في Microsoft" → "Person1 يعمل في Organization1"

Step 3: LLM Interaction

Send masked text to LLM → LLM processes without seeing real PII

Step 4: Unmask Response

LLM response with masked entities → Unmasked text with original PII

Example: "Person1 works at Organization1" → "احمد works at Microsoft"
