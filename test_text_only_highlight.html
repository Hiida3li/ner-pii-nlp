Advantages of Text-Only Highlighting:

  • Cleaner, more professional appearance
  • No overlapping issues with backgrounds
  • Better readability with less visual clutter
  • Maintains text flow without spacing disruptions
  • Works better with both light and dark themes
  • More accessible for users with visual sensitivities
  • Reduces cognitive load while still identifying entities
